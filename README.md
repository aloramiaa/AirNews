# AirNews - Modern News Platform

A modernized news aggregation platform built with **Next.js 15**, **Tailwind CSS**, and **Supabase**, featuring AI-powered article summaries, dual news feeds (India & International), real-time updates, and dark mode support.

## 🚀 Tech Stack

- **Framework**: Next.js 16.2.4 (App Router)
- **UI Framework**: React 19.2.4
- **Styling**: Tailwind CSS 4.2.4
- **Database**: Supabase (PostgreSQL)
- **Real-time**: Supabase Channels
- **Theme**: next-themes
- **Date Handling**: date-fns
- **Icons**: lucide-react
- **Package Manager**: pnpm v10+
- **Language**: TypeScript 5.9

## 📋 Features

✅ **Dual News Feeds** - India & International
✅ **AI Summaries** - Headlines and summaries generated by AI
✅ **Real-time Updates** - Breaking news notifications
✅ **Search** - Find articles by keyword
✅ **Dark Mode** - System preference + manual toggle
✅ **Responsive** - Mobile, tablet, desktop optimized
✅ **SEO Optimized** - Server-side rendering

## 🛠️ Getting Started

### Prerequisites
- Node.js 18+
- pnpm 10+ (`npm install -g pnpm`)

### Installation

```bash
# Install dependencies
pnpm install

# Set up environment (see .env.local)
# NEXT_PUBLIC_SUPABASE_URL and NEXT_PUBLIC_SUPABASE_ANON_KEY

# Run development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)

### Build for Production

```bash
pnpm build
pnpm start
```

## 📁 Project Structure

```
web/
├── app/                    # Next.js App Router pages
│   ├── page.tsx           # Home page with news feed
│   ├── layout.tsx         # Root layout
│   ├── articles/[id]/     # Article detail page
│   └── globals.css        # Global styling
├── components/            # React components
│   ├── Header.tsx        # Navigation & search
│   ├── ArticleCard.tsx   # Article preview
│   └── HomeClient.tsx    # Feed logic
├── lib/
│   ├── supabase.ts       # DB client
│   └── api.ts            # Data fetching
├── types/                # TypeScript definitions
└── public/               # Static assets
```

## 🔄 Key Features

### Home Page (`/`)
- News feed (India or International)
- Search box
- Article grid with load more
- Real-time breaking news alert

### Article Page (`/articles/[id]`)
- Full article content
- Tags and metadata
- Link to original source

### Dark Mode
- Auto-detects system preference
- Toggle via sun/moon button
- Persists across sessions
